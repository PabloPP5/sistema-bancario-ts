# Sistema Bancário em TypeScript

Projeto simples de sistema bancário em terminal, desenvolvido para treinar:

- Programação Orientada a Objetos
- TypeScript
- Node.js
- Entrada de dados via terminal

## Funcionalidades

- Depósito
- Saque
- Transferência
- Exibição de saldo

## Como rodar
```bash
npm install  
npx tsc  
node dist/sistemabancario.js
